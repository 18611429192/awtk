### 位操作扩展函数

#### &

> 位与运算。
----------------------------

##### 原型

```
&(n1,n2)
```

#### 示例

```
print(&(1, 1))
```

运行：

```
./bin/runFScript 'print(&(1, 2))'
```

输出：

```
0
```

#### |

> 位或运算。
----------------------------

##### 原型

```
|(1,2)
```

#### 示例

```
print(|(1, 2))
```

运行：

```
./bin/runFScript 'print(|(1, 2))'
```

输出：

```
3
```

#### ~

> 按位取反运算。
----------------------------

##### 原型

```
~(n1)
```

#### 示例

```
print(~(1))
```

运行：

```
./bin/runFScript 'print(iformat("0x%x", ~(1)))'
```

输出：

```
0xfffffffe
```
